# Meu Site

Este projeto foi desenvolvido como parte da disciplina de Desenvolvimento Web I.  
O site contém três páginas principais:

- **Página inicial**: apresenta a coleção de objetos tema.  
- **Página de detalhes**: mostra informações completas sobre um objeto.  
- **Página sobre o autor**: informações sobre mim, Clara.  

## Estrutura
- `index.html` → Página inicial  
- `detalhes.html` → Página de detalhes  
- `autor.html` → Página sobre o autor
